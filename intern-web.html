Intern Software Engineer - Web

Position Overview

We are looking for a motivated and enthusiastic Intern Web Developer to join our team. This is a great opportunity for someone looking to gain hands-on experience in web development, working alongside experienced developers on real-world projects. You will assist in designing, coding, and maintaining websites and web applications while learning best practices in front-end and/or back-end development.

Key Responsibilities

  • Assist in developing and maintaining responsive websites and web applications.
  • Write clean, efficient, and well-documented HTML, CSS, and JavaScript code.
  • Collaborate with the development team to implement new features and fix bugs.
  • Learn and apply modern frameworks/libraries such as React, Vue, Angular, or Node.js (depending on project needs).
  • Participate in code reviews and team meetings.
  • Test and debug applications across multiple browsers and devices.
  • Work with version control systems like Git.
  • Assist in integrating APIs and databases as needed.

Application Process

1

Submit Application

Complete our online application form. We review every application carefully to understand your skills and potential.

2

HR Screening

Selected candidates will have a 30-minute introductory call with our HR team.

3

Technical Interview

You'll meet with our development team for a 60-minute session including live coding exercises. We focus on understanding how you think and learn.

4

IQ Test & Finalizatation

Our hiring team makes final decisions within 5-7 business days. Successful candidates receive a detailed offer.

Requirements

Qualifications & Skills

  • Currently pursuing or recently completed a degree in Computer Science, Web Development, or a related field (or equivalent experience).
  • Basic understanding of HTML, CSS, and JavaScript.
  • Familiarity with front-end frameworks (React, Vue, Angular) or back-end technologies (Node.js, Python, PHP) is a plus.
  • Eagerness to learn and adapt to new technologies.
  • Strong problem-solving skills and attention to detail.
  • Ability to work both independently and in a team environment.
  • Basic knowledge of Git and version control is preferred.

Nice to Have (But Not Required)

  • Experience with databases (SQL, MongoDB, Firebase).
  • Knowledge of UI/UX principles.
  • Exposure to cloud platforms (AWS, Azure, Google Cloud).
  • Previous internship or personal project experience in web development.
